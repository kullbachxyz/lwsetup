# lwsetup

Script to setup Librewolf with the Arkenfox user.js, as well as some custom user.js overrides. Also installs the following extensions:

  - uBlock Origin
  - Decentraleyes
  - I Still Don't Care About Cookies
  - Vimium

## Usage

```sh
sh lwsetup.sh
```

## Why?
I was tired of setting up Librewolf every time I newly installed a system, so i made this setup script.
